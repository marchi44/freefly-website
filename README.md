# Komparch
FreeFly is a full-stack web application designed to streamline travel planning by enabling users to search for real-time flight data, filter and compare options, and manage bookings within an integrated platform.

The system was developed using React and Tailwind CSS for a responsive and modern user interface, alongside a backend built with Express.js and MongoDB for data management and user authentication.

Key features include:
  Real-time flight search powered by external API integration
  Advanced filtering and sorting of flight results
  User authentication system (registration & login)
  Simulated wallet system for managing virtual funds
  Ticket booking and management functionality
  Fully responsive UI built with Tailwind CSS
  Deployment using Vercel and Render
